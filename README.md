(A)請寫一個程式把裡面的字串反過來。
```py
def reverse(txt):
  return txt[::-1]

print(reverse("junyiacademy"))
```

(B)請寫一個程式把裡面的字串,每個單字本身做反轉,但是單字的順序不變。
```py
def reverseWord(txt):
  word_list = txt.split()
  result = ''
  for word in word_list:
    result += reverse(word) + ' '
  return result

print(reverseWord("flipped class room is important"))
```

# Q2

請寫一個程式,Input 是一個數字,Output 是從 1 到這個數字,扣除掉所有 3 的倍數以及 5 的倍數,但是需要保留同時是 3 和 5 的倍數的總數字數。

```py
def remove(num):
  num3 = num//3 
  num5 = num//5 
  num15 = num//15 
  return num - num3 - num5 + 2*(num15)

print(remove(30))
```

Q3: 房間裡有三個袋子,一個只裝鉛筆,一個只裝原子筆,第三個有鉛筆也有原子筆。袋子是不透明的,單從袋子的外表上看不出任何差異,你不知道哪個袋子裝什麼。除了袋子上各貼了一個標示("鉛筆"、"原子筆"、"混和"),且標示都是錯的(e.g. 標示鉛筆的袋子可能是混和的或是只裝原子筆)。你只能選一個袋子,拿出裡面一支筆,看是鉛筆還是原子筆,然後你要推論出這三個袋子分別的情況。請列出你的作法,以及解釋為什麼這樣可以找到答案。

## Ans

只要直接打開「混合」的袋子，就可以確定其他袋子狀態

###  如果從「混合」拿出的是「原子筆」，則
- 標記「混合」的袋子一定會是「原字筆」，因為
  - 標示一定是恰巧錯誤的，所以該袋不可能是「混合」，只可能是「原字筆」
- 標記「鉛筆」的袋子一定是「混合」，因為
  1. 標記一定恰巧錯誤，所以該袋子只可能是「原字筆」 or 「混合」
  2. 標記「混合」的袋子確定是「原字筆」，所以該袋子只可能是「混合」
- 標記「原字筆」的袋子一定是「鉛筆」，因為
  1. 標記一定恰巧錯誤，所以該袋子只可能是「鉛筆」 or 「混合」
  2. 標記「鉛筆」的袋子確定是「混合」，所以該袋子只可能是「鉛筆」

### 如果從「混合」拿出的是鉛筆，則
- 標記「混合」的袋子一定是「鉛筆」，因為
  - 標記一定是錯誤的，所以該袋不可能是「混合」，可能是「原字筆」 
- 標記「原字筆」的袋子一定是「混合」，因為
  1. 標記一定恰巧錯誤，所以該袋子只可能是「鉛筆」 or 「混合」
  2. 標記「混合」的袋子確定是「鉛筆」，所以該袋子只可能是「混合」
- 標記「鉛筆」的袋子一定是「原字筆」，因為
  1. 標記一定恰巧錯誤，所以該袋只可能是「混合」 or 「原字筆」
  2. 標記「原子筆」的袋子確定是「混合」，所以該袋子只可能是「原字筆」

> 重要的大前提：「袋子上的標示都是錯的」（不會有剛好標示正確的狀況）

Q4：有三個人一起到迪士尼遊玩,中午肚子餓了,去餐廳點了一份現在最夯的冰雪奇緣雙人組,要價 900 元,付錢後,服務生發現今天套餐大特價,只要 750 元,因此服務生應該退還 150 元給這三個人,但是這位服務生一時鬼迷心竅,決定暗槓 60元,只退了 90 元給這三個遊客。
那麼:三人各出 300 元 - 服務生還給他們一人 30 元 = 三人各出 270 元。270元 × 3 人+ 服務生私吞的 60 元 = 810 + 60 = 870 !? 怎麼不是 900 元呢?還有 30 元去哪了呢?請用敘述的方式,儘量清楚解釋問題出在哪裡。

## Ans

問題出在「三人各出 270 元 × 3 人+ 服務生私吞的 60 元」這個算式。
該算式的問題在於「服務生私吞的 60 元」是個`重複計算`的項目：
- 每個人付出的成本 270 元，就已經包含了商品價值 250 元（ 750 / 3 人）＋服務生私吞的 20 元（ 60 / 3 人），因此「服務生私吞的 60 元」在該算式中多被計算了一次
- 正確的算式應該要是「270元 （商品價值250＋服務生私吞30）× 3 人+ 找的 90元 」才會剛好等於付出的 900 元
